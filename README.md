# antd-custom-table
based on ant-design Table, extend header contextMenu settings , and auto fit scroll-x  scoll-y.

## how to use it

> <CustomTable<br/>
>  name={'my-table-1'} <br/>
>  className={'auto-scroll-y'} <br/>
>  columns={this.tableColumns} <br/>
>  dataSource={tableDataSource} <br/>
>  scroll={{x: true, y: true}} <br/>
> />
